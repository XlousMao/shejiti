**1.2 考虑下述选择排序算法：**

**算法ModSelectSort**

**输入：n个不等的整数的数组A\[1..n\]**

**输出：按递增次序排序的A**

**1\. for i←1 to n-1 do**

**2\. for j←i+1 to n do**

**3\. if A\[j\]<A\[i\] then A\[j\]↔A\[i\]**

**问：**

1.  **最坏情况下该算法做多少次比较运算？**
2.  **最坏情况下该算法做多少次交换运算？这种情况在什么输入条件下发生？**
3.  **最坏情况下的比较次数是：**
4.  **当输入的n个数彼此不等且按递降次序排列时，比较次数是n(n-1)/2。每次比较后都发生交换，因此最坏情况下的交换次数也是n(n-1)/2。**

#include <iostream> //VS2010

using namespace std;

int n,A\[10\];

void ModSelectSort()

{

for(int i=1;i<=n-1;i++)

for(int j=i+1;j<=n;j++)

if(A\[j\]<A\[i\])

swap(A\[j\],A\[i\]);

}

int main()

{

while(cin>>n)

{

for(int i=1;i<=n;i++)

cin>>A\[i\];

ModSelectSort();

for(int i=1;i<n;i++)

cout<<A\[i\]<<",";

cout<<A\[n\]<<endl;

}

return 0;

}
